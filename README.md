# E-commerce Admin Dashboard

Dashboard administrativo desarrollado con Angular.

## Descripción

E-commerce Admin Dashboard es un panel administrativo moderno diseñado para gestionar productos, usuarios y pedidos de una tienda virtual.

El proyecto fue desarrollado con Angular y cuenta con una interfaz moderna, responsive e interactiva enfocada en la experiencia de usuario y diseño UI/UX.

## Características

- Login UI moderno
- Sidebar interactivo
- Dashboard responsive
- Gráficas con Chart.js
- Gestión de productos
- Gestión de usuarios
- Gestión de pedidos
- Diseño moderno UI/UX
- Navegación dinámica entre secciones

## Tecnologías utilizadas

- Angular
- TypeScript
- CSS
- Chart.js
- Bootstrap Icons

## Objetivo del proyecto

Desarrollar un dashboard administrativo funcional y visualmente atractivo para un e-commerce, permitiendo gestionar información de manera organizada mediante una interfaz moderna e intuitiva.
